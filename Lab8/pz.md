## 1. Основная цель программы

Программа реализует **игру «Угадай число»**, где:

* Компьютер **загадывает случайное число** в зависимости от выбранного уровня сложности.
* Пользователь **пытается угадать число** за ограниченное количество попыток.
* После каждой попытки программа подсказывает, больше или меньше введённое число загаданного.
* Пользователь может **играть несколько раундов**, пока сам не решит выйти.

---

## 2. Структура переменных

```pascal
var
  b, uroven, chislo, popytki, number, max: integer;
```

* `b` — флаг продолжения игры (`1` — играть, `0` — выйти).
* `uroven` — выбранный пользователем уровень сложности (от 1 до 4).
* `chislo` — загаданное компьютером число.
* `popytki` — счётчик текущей попытки пользователя.
* `number` — число, введённое пользователем в текущей попытке.
* `max` — максимальное число попыток для выбранного уровня.

---

## 3. Инициализация игры

```pascal
randomize;
```

* Инициализирует генератор случайных чисел, чтобы каждый раз программа генерировала разные числа.

Приветствие пользователя:

```pascal
writeln('###############################');
writeln('##### Игра "Угадай число" #####');
writeln('###############################');
```

---

## 4. Главный игровой цикл

```pascal
repeat
  writeln;
  uroven := 0;
  popytki := 0;
```

* `repeat ... until b=0` — основной цикл, который повторяет игру, пока пользователь хочет продолжать.
* Перед каждой игрой сбрасываются переменные `uroven` и `popytki`.

---

## 5. Выбор уровня сложности

```pascal
while (uroven < 1) or (uroven > 4) do
begin
  writeln('Выберите уровень сложности:');
  writeln('1 - Числа от 1 до 10, 3 попытки');
  writeln('2 - Числа от 1 до 50, 5 попыток');
  writeln('3 - Числа от 1 до 100, 8 попыток');
  writeln('4 - Числа от 1 до 1000, 15 попыток');
  readln(uroven);
end;
```

* Цикл **защищает от некорректного ввода**: пользователь может выбрать только 1–4.
* Каждый уровень задаёт **диапазон чисел и количество попыток**.

---

## 6. Генерация загаданного числа и установка максимальных попыток

```pascal
if uroven =1 then begin chislo:=random(10)+1; max:=3 end; 
if uroven =2 then begin chislo:=random(50)+1; max:=5 end;
if uroven =3 then begin chislo:=random(100)+1; max:=8 end;
if uroven =4 then begin chislo:=random(1000)+1; max:=15 end;
```

* `random(N)` генерирует число от 0 до N-1, поэтому добавляем `+1`, чтобы диапазон начинался с 1.
* `max` — максимальное число попыток для текущего уровня.

---

## 7. Основной цикл угадывания

```pascal
repeat
  inc(popytki);
  write('Попытка ',popytki,' Ваше число? ');
  read(number);

  if number>chislo then writeln('   Я загадал число поменьше') 
  else if number<chislo then writeln('   Я загадал число побольше') 
  else writeln('Вы угадали!')
until (chislo=number) or (popytki=max);
```

* `repeat ... until` выполняется, пока пользователь **не угадает число** или **не закончатся попытки**.
* `inc(popytki)` увеличивает номер текущей попытки.
* Программа подсказывает:

  * `"число поменьше"`, если введённое число больше загаданного.
  * `"число побольше"`, если введённое число меньше.
  * `"Вы угадали!"`, если число совпало.

---

## 8. Сообщение о результате

```pascal
if (chislo<>number) and (popytki=max) then 
  writeln('К сожалению, Вам не удалось отгадать число. Я загадал ',chislo);
```

* Если попытки закончились, а пользователь не угадал, программа сообщает **загаданное число**.

---

## 9. Предложение сыграть ещё раз

```pascal
writeln('Хотите сыграть еще раз? Если да, введите 1, если нет, то 0');
readln(b);
```

* Если пользователь вводит `1`, игра повторяется.
* Если `0`, программа завершает работу.

---

## 10. Итоговая логика программы

1. Приветствие пользователя и выбор уровня сложности.
2. Компьютер загадывает число.
3. Пользователь угадывает число за ограниченное количество попыток.
4. Программа подсказывает, больше или меньше введённое число загаданного.
5. После завершения раунда можно сыграть снова.
