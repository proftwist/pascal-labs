# pascal-labs

Этот репо я буду использовать, преимущественно для освоения гита. Буду заливать сюда лабы на паскале, пробовать коммитить. 
Программы откомпилированы под макось, кодировка русских текстов — UTF-8.

* Lab1 — работа с целочисленными переменными. Чтение ввода с клавиатуры. 
* Lab2 — работа с циклами. 
* Lab3 — работа с простейшей логикой. 
* Lab4 — работа с массивами. 
* Lab5 — работа со строками. 
* Lab6 — работа с процедурами, раскрашивание текста, сложная логика. 
* Lab7 — работа с объектами в оперативной памяти.
