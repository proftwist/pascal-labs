# pascal-labs

Программы откомпилированы под макось, кодировка русских текстов — UTF-8.
Можете перекодировать исходники в кодировку своей ОС и откомпилировать под неё. 

Код снабжён комментариями, в каждой папке с лабой лежит пояснительная записка pz.md.

За основу взяты лабораторные работы, выполненные мной на первом курсе специальности «Прикладная информатика в экономике» в 2004 году.

* Lab1 — работа с целочисленными переменными. Чтение ввода с клавиатуры.
* Lab2 — работа с циклами.
* Lab3 — работа с простейшей логикой.
* Lab4 — работа с массивами.
* Lab5 — работа со строками.
* Lab6 — работа с процедурами, раскрашивание текста, сложная логика.
* Lab7 — работа с объектами в оперативной памяти.
* Lab8 — работа с рандомизацией, простенькая игра.
* Lab9 — работа с файлами.
* Lab10 — работа с проверками на допустимость, цветами текста, бипером.
* Kurs — курсовая работа, большая многомодульная программа, читающая данные из файла и пишущая в него.
